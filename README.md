## Projects

### 1. Disease Diagnosis: Multimodal CNN-LSTM for Agriculture
A multimodal deep learning system combining image data and environmental context to provide highly accurate disease diagnosis for crops. The system integrates CNNs for visual analysis with LSTMs for weather data modeling to resolve ambiguity in plant pathology.

Technologies: Python, PyTorch, TensorFlow, OpenCV, Pandas, NumPy
